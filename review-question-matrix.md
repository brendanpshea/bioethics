# Closing Review Question Matrix

This file reserves a distinct job for the closing review slide in each of the 24 lesson-parts.

## Working Rules

- Recall questions retrieve definitions, distinctions, mechanisms, or framework maps from that lesson-part only.
- Apply questions use a new scenario, not the lesson's opening hook, running patient, paired case study, or existing Thought Question stem.
- Debate questions reserve one current controversy lane per lesson-part so later lessons do not inherit a topic that was already spent earlier.
- The closing slide should synthesize the whole lesson-part. It should not simply run the body discussion prompt one more time.
